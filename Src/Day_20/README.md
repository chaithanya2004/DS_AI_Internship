# 📊 Customer Analytics – EDA Mini Project

## 📌 Project Overview

This mini project focuses on performing Exploratory Data Analysis (EDA) on a Customer Analytics dataset to uncover patterns in customer behavior.

The goal of this project is to clean the dataset, analyze distributions, identify trends, and generate actionable business insights before applying machine learning models.

Exploratory Data Analysis helps in understanding the data deeply and preparing it for predictive modeling.

---

## 📂 Dataset Used

File: `customer_analytics.csv`

The dataset contains customer-related information such as:
- Age
- Gender
- Annual Income
- Spending Score
- Other behavioral features

---

## 🛠 Tools & Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 🔎 EDA Steps Performed

1️⃣ Data Loading and Inspection  
- Used `head()`, `info()`, and `describe()` to understand structure  

2️⃣ Missing Value Analysis  
- Identified and handled missing values  

3️⃣ Duplicate Record Detection  
- Checked and removed duplicate entries  

4️⃣ Univariate Analysis  
- Histograms to understand distribution  
- Boxplots to detect outliers  

5️⃣ Bivariate Analysis  
- Compared spending behavior across Gender  
- Analyzed relationship between Income and Spending Score  

6️⃣ Correlation Analysis  
- Generated correlation matrix  
- Identified strongly related numerical features  

---

## 📈 Key Insights

- Spending Score varies significantly across different income groups.
- Some numerical features contain outliers that may impact model accuracy.
- Income and Spending Score show noticeable correlation patterns.
- Customer segmentation can be performed using clustering techniques.
